Overview

  • This app demonstrates core and commonly used features of DataFormsJS and how to display data from JSON Web Services and GraphQL and using only markup (HTML and Templating).
  • Common plugins and components used include filter and sorting. Use the input controls to type and quickly filter data on screen and click on column headers to sort table rows based on data type.
  • This app is written in several different front-end technologies to show different options of using DataFormsJS (Handlebars, Vue, React, Web Components, and JavaScript).
  • When using the core Framework (Handlebars, Vue, and JavaScript) scripts are dynamically downloaded only when first used which allows for greater performance and a smaller site when the first page loads.
  • The React example uses Babel Standalone with DataFormsJS React Components and JSX Templates for the Pages.
  • This app uses data from the GeoNames Database, flags from Semantic UI, mapping from Leaflet with OpenStreeMap, and additional third-party components on Handlebars and GraphQL demos including jQuery and the Chosen jQuery plugin.
